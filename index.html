Configurar chave de API

O BioScan usa o Claude (Anthropic) para analisar as imagens. Você precisa de uma chave de API, que pode ser criada gratuitamente em console.anthropic.com.

⚠ A chave é salva apenas localmente neste dispositivo e nunca enviada para servidores do app. Recomendamos configurar um limite de gastos na conta Anthropic.
